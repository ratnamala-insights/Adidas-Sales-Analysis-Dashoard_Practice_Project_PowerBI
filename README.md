**Project Overview**
This repository contains a practice data analysis project built using Power BI. The project explores sales performance using a publicly available Adidas sales dataset and demonstrates how interactive dashboards can be used to analyze business metrics such as revenue, profit, regional performance, and product category trends.
The goal of this exercise was to practice data visualization, dashboard design, and business insight generation using Power BI.

**Objective**

The primary objective of this practice project is to:

Understand how to transform raw sales data into interactive business insights

Practice data modeling and visualization techniques in Power BI

Analyze sales performance across time, regions, products, and retailers

Build an executive-style dashboard that summarizes key business metrics

**Tools & Technologies Used**

Power BI – Dashboard creation and visualization

Microsoft Excel – Data source

DAX (Data Analysis Expressions) – Measures and calculations

**Dataset**

The dataset used in this project was downloaded from kaggle which contains Adidas sales information such as:

Retailer

Retailer ID

Invoice Date

Region, State, City

Product

Price per Unit

Units Sold

Total Sales

Operating Profit

Operating Margin

Sales Method

The dataset is used only for analysis and visualization practice purposes.

**Key Insights**

Some observations from the dashboard analysis include:

Men’s Street Footwear generates the highest sales among product categories.

West Gear and Foot Locker are among the top-performing retailers.

Sales show noticeable seasonal fluctuations, with peak performance during mid-year months.

Certain states contribute significantly higher revenue, indicating strong regional demand.

A healthy operating margin of over 40% suggests strong profitability across the product mix.
